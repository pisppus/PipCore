PipCore — это легковесный, высокопроизводительный слой аппаратных абстракций (HAL) и системное ядро, разработанное специально для микроконтроллеров.

Ядро обеспечивает эффективный доступ к платформе, работу с GPIO и регулировку подсветки через LEDC PWM. Оно содержит асинхронный двухбуферный DMA-драйвер для дисплеев. Движок спрайтов поддерживает отсечение по маске, быстрые 32-битные операции с аппаратным байт-свопом и программное альфа-смешивание в пространстве RGB565. Кроме того, библиотека содержит неблокирующую OTA-службу, проверкой подписи манифеста и контролем целостности по SHA256. Встроенный симулятор для Windows обеспечивает отрисовку кадров, эмуляцию ввода, сохранение PNG-скриншотов и запись MP4-видео.
