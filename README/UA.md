PipCore — це легкий, високопродуктивний рівень апаратних абстракцій (HAL) та системне ядро, розроблене спеціально для мікроконтролерів.

Ядро забезпечує ефективний доступ до платформи, роботу з GPIO та регулювання підсвічування через LEDC PWM. Воно містить асинхронний двобуферний DMA-драйвер для дисплеїв. Движок спрайтів підтримує відсікання за маскою, швидкі 32-бітні операції з апаратним байт-свопом та програмне альфа-змішування в просторі RGB565. Крім того, бібліотека містить неблокуючу OTA-службу з перевіркою підпису маніфесту та контролем цілісності за SHA256. Вбудований симулятор для Windows забезпечує відтворення кадрів, емуляцію введення, збереження PNG-скріншотів та запис MP4-відео.
